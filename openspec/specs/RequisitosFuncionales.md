# Requisitos Funcionales

## Objetivo

Este documento define los requisitos funcionales del Sistema de Gestión de Préstamos de Equipos. Cada requisito representa una capacidad que el sistema deberá proporcionar para satisfacer las necesidades de los usuarios.

---

## RF-01 - Autenticación de usuarios

El sistema deberá permitir que los usuarios inicien sesión mediante un nombre de usuario y contraseña válidos.

---

## RF-02 - Gestión de usuarios

El sistema deberá permitir registrar, consultar, modificar y desactivar usuarios autorizados.

---

## RF-03 - Gestión de equipos

El sistema deberá permitir registrar, consultar, actualizar y desactivar equipos disponibles para préstamo.

---

## RF-04 - Consulta de disponibilidad

El sistema deberá mostrar el estado actual de cada equipo (Disponible, Prestado o En mantenimiento).

---

## RF-05 - Registro de préstamos

El sistema deberá permitir registrar préstamos de equipos asociados a un usuario.

---

## RF-06 - Registro de devoluciones

El sistema deberá permitir registrar la devolución de un equipo prestado, actualizando automáticamente su disponibilidad.

---

## RF-07 - Consulta de historial

El sistema deberá permitir consultar el historial de préstamos y devoluciones realizados.

---

## RF-08 - Búsqueda y filtrado

El sistema deberá permitir buscar equipos y préstamos utilizando diferentes criterios de búsqueda.

---

## RF-09 - Validación de disponibilidad

El sistema deberá impedir registrar préstamos sobre equipos que no se encuentren disponibles(mantenimiento, prestados o no activos).

---

## RF-10 - Gestión de sesiones

El sistema deberá permitir cerrar la sesión del usuario autenticado de forma segura.
